# 内容工厂

一条 AI 内容生产流水线。当前以「小红书宠物号」为**低风险试验田**跑通工作流；长期是**打造个人 IP 的底座**，也是潜在可产品化的资产（卖工作流 / 模板）。

## 定位（别忘了）

- 资产是**工作流**，不是某个账号。**别爱上号，要爱上流水线。**
- 成功指标：单条可发布内容的**人手工时**（越低越好），不是粉丝数。号的数据只是判断产出质量的传感器。

## 流水线

选题（即刻 / TrendRadar）→ 生图（nanobanana）→ 视频（veo3）→ 配音（noiz）→ 编排 / 发布（n8n / 扣子）

**价值密度**（力气别平均花）：

- 生成 = 已商品化，接通即可
- 选题 = 决定约 60% 成败
- 编排 + 一致性 = 工程优势 & 可产品化部分 ← 重点

## 当前实验（限时 4 周）

- **目标**：跑通全自动流水线，产出约 20 条
- **Kill 标准**：单条人手工时降不到可接受线（如 <15 min）就停——认知照样赚到
- **约束**：别挤占 B 轨（测试工具，更接近变现）；每周分配 ___ h（待你定）
- **复用优先**：先 fork 现成 n8n / 扣子 AI 内容工厂模板，别从零搭
